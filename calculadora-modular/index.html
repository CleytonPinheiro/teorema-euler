Calculadora de Potências Modulares

Aplica o Teorema de Euler para reduzir expoentes grandes quando mdc(a, n) = 1, e exponenciação rápida nos demais casos.

📐

Teorema de Euler

Enunciado

Se a e n são inteiros positivos com mdc(a, n) = 1, então:

aφ(n) ≡ 1 (mod n)

onde φ(n) é a função totiente de Euler — o número de inteiros entre 1 e n que são coprimos com n.

Função Totiente φ(n)

Para calcular φ(n), usamos a fórmula com a fatoração de n em primos:

φ(n) = n · ∏p|n (1 − 1/p)
φ(7) = 6 (7 é primo)
φ(10) = 4 {1,3,7,9}
φ(12) = 4 {1,5,7,11}
φ(1) = 1 por convenção

Exemplo Passo a Passo

Calcule 31000 mod 7

  1. 1
    Verificar coprimalidade:
    mdc(3, 7) = 1 ✓  → Euler se aplica
  2. 2
    Calcular φ(7):
    7 é primo → φ(7) = 7 − 1 = 6
  3. 3
    Reduzir o expoente:
    1000 mod 6 = 4
    pois 1000 = 166×6 + 4
  4. 4
    Aplicar o teorema:
    31000 ≡ 34 (mod 7)
  5. 5
    Calcular 34 mod 7:
    34 = 81 = 11×7 + 4
    Resultado: 4

Quando não se aplica?

Se mdc(a, n) ≠ 1, o teorema de Euler não garante a redução. Nesse caso a calculadora usa a exponenciação modular rápida (square-and-multiply), que é eficiente mesmo para expoentes gigantescos.
